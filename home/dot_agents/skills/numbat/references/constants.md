# Mathematical and Physical Constants

## Mathematical Constants

| Identifier | Description |
|---|---|
| `pi`, `π` | Pi (3.14159...) |
| `τ` | Tau = 2π |
| `e` | Euler's number (2.71828...) |
| `golden_ratio`, `φ` | Golden ratio (1.61803...) |

## Named Numbers

### Large numbers

| Identifier | Value |
|---|---|
| `hundred` | 100 |
| `thousand` | 1000 |
| `million` | 10^6 |
| `billion` | 10^9 |
| `trillion` | 10^12 |
| `quadrillion` | 10^15 |
| `quintillion` | 10^18 |
| `googol` | 10^100 |

### Unicode fractions

`½`, `⅓`, `⅔`, `¼`, `¾`, etc.

### Colloquial

`quarter`, `half`, `semi`, `double`, `triple`, `dozen`

## Physical Constants

| Description | Identifier | Dimension |
|---|---|---|
| Speed of light in vacuum | `speed_of_light`, `c` | `Velocity` |
| Newtonian gravitational constant | `gravitational_constant`, `G` | `Force × Length² / Mass²` |
| Standard gravity | `gravity`, `g0` | `Acceleration` |
| Planck constant | `planck_constant`, `ℎ` | `Mass × Length² / Time` |
| Reduced Planck constant | `h_bar`, `ℏ` | `Mass × Length² / Time` |
| Electron mass | `electron_mass` | `Mass` |
| Elementary charge | `elementary_charge`, `electron_charge` | `ElectricCharge` |
| Magnetic constant | `magnetic_constant`, `µ0`, `mu0` | `Force / Current²` |
| Electric constant | `electric_constant`, `ε0`, `eps0` | `Capacitance / Length` |
| Bohr magneton | `bohr_magneton`, `µ_B` | `Energy / MagneticFluxDensity` |
| Fine structure constant | `fine_structure_constant`, `alpha`, `α` | `Scalar` |
| Proton mass | `proton_mass` | `Mass` |
| Neutron mass | `neutron_mass` | `Mass` |
| Avogadro constant | `avogadro_constant`, `N_A` | `1 / AmountOfSubstance` |
| Boltzmann constant | `boltzmann_constant`, `k_B` | `Energy / Temperature` |
| Stefan-Boltzmann constant | `stefan_boltzmann_constant` | `Power / (Area × Temperature⁴)` |
| Ideal gas constant | `gas_constant`, `R` | `Energy / (AmountOfSubstance × Temperature)` |
| Planck length | `planck_length` | `Length` |
| Planck mass | `planck_mass` | `Mass` |
| Planck time | `planck_time` | `Time` |
| Planck temperature | `planck_temperature` | `Temperature` |
| Planck energy | `planck_energy` | `Energy` |
| Bohr radius | `bohr_radius`, `a0` | `Length` |
| Rydberg constant | `rydberg_constant` | `Wavenumber` |
