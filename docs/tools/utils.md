# General Utils

## Poppler Utils

Exacutables provided by package `poppler-utils`, a set of CLI tools using
[Poppler](https://poppler.freedesktop.org/):

- `pdfattach`
- `pdfdetach`
- `pdffonts`
- `pdfimages`
- `pdffonts`
- `pdfseparate`
- `pdfsig`
- `pdftocairo`
- `pdftohtml`
- `pdftoppm`
- `pdftops`
- `pdftotext`
- `pdfunite`
