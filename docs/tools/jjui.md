# [Jujutsu UI](https://docs.jj-vcs.dev)

Available themes are configured as chezmoi externals from
[tinted-jjui](https://github.com/vic/tinted-jjui):

- `base16-solarized-dark`
- `base16-gruvbox-dark`

## Custom Commands

| Key | Command | Description |
|-----|---------|-------------|
| `p` | push with created bookmark | Push with `-c` flag to create bookmark |
| `u` | update time | Update author timestamps for a branch (prompts with closest pushable bookmark) |
