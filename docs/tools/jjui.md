# [Jujutsu UI](https://jj-vcs.github.io)

TUI for interacting with Jujutsu VCS.

Available themes are configured as chezmoi externals from
[tinted-jjui](https://github.com/vic/tinted-jjui):

- `base16-solarized-dark`
- `base16-gruvbox-dark`

## Leader Keys

Custom leader keys configured in `~/.config/jjui/config.toml`:

- `\k`: Move revision up
- `\j`: Move revision down
- `\a`: New change after
- `\e`: Edit checked files
- `\m`: Metaedit update timestamps
- `\A`: Abandon branch from trunk

### Community Contribs

- `\t`: Tug bookmark to current
- `\B`: Rebase all to trunk
- `\R`: Save revset as alias
