# [Jujutsu UI](https://docs.jj-vcs.dev)

Available themes are configured as chezmoi externals from
[tinted-jjui](https://github.com/vic/tinted-jjui):

- `base16-solarized-dark`
- `base16-gruvbox-dark`

## Custom Commands

| Key | Description |
| -------------- | --------------- |
| `P` | Push revision with created bookmark |
| `N` | New revision after focused and before descendants |
