# Jujutsu (jj)

Jujutsu is a Git-compatible VCS that's simple and powerful.

## Aliases

Custom aliases configured in `~/.config/jj/config.toml`:

- `ui`: Launches jjui TUI
- `metaedit-update-timestamps`: Updates author and committer timestamps for a revision
- `abandon-branch-from-trunk`: Abandons revisions in the branch not in trunk

### Community Contribs

- `tug`: Moves the bookmark from the head of the current branch to the current revision
- `rebase-all`: Rebases all mutable changes onto trunk</content>
